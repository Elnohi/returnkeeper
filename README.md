# ReturnKeeper

Never miss a return deadline again.

## The Problem

Americans lose over $50 billion annually on forgotten returns and expired warranties. ReturnKeeper solves this by automatically tracking deadlines from your receipts.

## Features

- 📸 Snap any receipt
- ⏰ Auto-detects return deadline (30/60/90 days)
- 🔔 Alerts you 3 days before
- 🛡️ Tracks warranties too
